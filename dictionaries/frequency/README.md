# Frequency Dictionaries

Drop Yomichan-format frequency dictionary ZIP files here. They'll be auto-imported on backend startup alongside regular dictionaries.

Supported format: Yomichan frequency dictionaries (e.g., JPDB, BCCWJ, VN freq lists)

Example: `[JA Freq] JPDB (2024).zip`

Multiple sources supported — each shows as a separate option in the frequency filter dropdown.
